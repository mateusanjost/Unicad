<h1>INSTRUÇÕES</h1>

Crie um banco de dados e o configure no .env:


DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=nomedobanco
DB_USERNAME=root
DB_PASSWORD=

Demonstração: https://www.youtube.com/watch?v=t23fHBMAoJY&feature=youtu.be

Como usar:

execute na pasta do projeto:

composer install
npm install
npm dev
php artisan key:generate
php artisan migrate
npm run watch
php artisan serve
